# SearxGo Browser App

SearxGo 🛡️
​O SearxGo é um navegador Android focado em privacidade, desenvolvido com Flutter, projetado para oferecer uma experiência de navegação limpa, rápida e sem rastreadores.
​✨ Características
​Privacidade em Primeiro Lugar: Navegação anônima e sem rastreamento de dados.
​Design Minimalista: Interface moderna com foco na facilidade de uso.
​Barra de Busca Integrada: Experiência otimizada com acesso rápido a recursos de pesquisa privada.
​Leve: Otimizado para não consumir recursos desnecessários do seu dispositivo.

​🚀 Como Compilar
​

### Este projeto utiliza Flutter. Para rodar no seu dispositivo Android:
Pré-requisitos
Flutter SDK instalado.
Android Studio ou VS Code configurado.
Modo Desenvolvedor ativado no seu celular Android.
Passos
Clone o repositório:

git clone [URL_DO_SEU_REPOSITORIO]
cd searxgo_browser_app

Obtenha as dependências

Conecte seu dispositivo e execute:
flutter run


🛠️ Tecnologias Utilizadas
Flutter: Framework para UI nativa.
[Kotlin/Android]: Configuração nativa para performance e integração com o sistema operacional.
Searx: Motor de busca utilizado para garantir a privacidade.
